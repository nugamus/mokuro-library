# Using the Reader

The reader is Mokuro Library's core feature - a powerful, customizable interface for reading manga with integrated OCR text selection and editing capabilities.

## Opening the Reader

### From Library View

1. Click on any series card
2. On the series page, click a volume thumbnail
3. The reader opens to the first page (or last read page)

### From Series Page

1. Navigate to a series detail page
2. Click "**Read**" on any volume
3. Or use the "**Continue Reading**" button for in-progress volumes

## Layout Modes

Mokuro Library offers three distinct reading layouts to match your preferences.

### 1. Single Page Mode

**Default mode** - One page at a time.

**Best For:**
- Most manga reading
- Mobile devices
- Focused reading
- Language learning (OCR selection)

**Features:**
- Full page visibility
- Easy OCR text selection
- Works on all screen sizes
- Quick page navigation

**Navigation:**
- Click left/right sides
- Arrow keys (‚Üê ‚Üí)
- Swipe on touch devices

![Single Page Mode](/single-page-mode.webp)
*Single page with OCR text overlay*

### 2. Double Page (Spread) Mode

**Side-by-side pages** simulating an open book.

**Best For:**
- Desktop/tablet reading
- Double-page spread artwork
- Traditional manga experience
- Larger screens

**Features:**
- Two pages displayed simultaneously
- Configurable reading direction
- Page offset for proper spreads
- Immersive reading experience

![Double Page Settings](/double-page-settings.webp)
*Configure double-page spread options*

**Additional Settings:**

**Reading Direction:**
- **Right-to-Left** - Traditional manga (default)
- **Left-to-Right** - Western comics

**Dual-Page Offset:**
- **Odd Start** - First page alone, then pairs
- **Even Start** - Pairs from page 1
- Use to align spreads correctly

::: tip Spread Alignment
If double-page artwork looks split incorrectly, toggle the offset setting to realign the spreads.
:::

### 3. Vertical Scroll Mode

**Infinite scrolling** - All pages in one continuous column.

**Best For:**
- Webtoon-style reading
- Continuous flow
- Mobile reading
- Long reading sessions

**Features:**
- Seamless page transitions
- Natural scrolling
- No page breaks
- Smooth reading flow

![Vertical Scroll Mode](/vertical-scroll-mode.webp)
*Vertical scrolling through entire volume*

**Behavior Changes:**
- Navigation buttons hidden
- Keyboard arrows disabled
- Scroll to navigate
- Touch-optimized

**Scrolling:**
- Mouse wheel
- Trackpad gestures
- Touch/swipe
- Scrollbar

::: info Note
Reading direction and page offset settings don't apply in vertical mode since all pages scroll in one direction.
:::

## OCR Text Selection

The killer feature - select and copy Japanese text directly from manga pages.

### How It Works

**Mokuro's OCR Technology:**
- Pre-processed text detection
- Character recognition
- Selectable text overlays
- Dictionary-friendly output

**Using Text Selection:**

1. **Hover** over a text bubble
   - OCR overlay highlights
   - Text becomes visible

2. **Click** to select text
   - Text is highlighted
   - Ready to copy

3. **Copy** the text
   - Right-click ‚Üí Copy
   - Ctrl+C (Cmd+C on Mac)
   - Paste into dictionaries or notes

![OCR Text Selection](/ocr-selection-placeholder.svg)
*Hover over text to reveal OCR overlay*

### Language Learning Workflow

**Perfect for Japanese learners:**

1. **Read** manga naturally
2. **Hover** over unfamiliar words
3. **Select and copy** the text
4. **Paste** into:
   - [Jisho.org](https://jisho.org) - Dictionary
   - [DeepL](https://www.deepl.com) - Translation
   - Yomichan - Browser extension
   - Anki - Flashcard app

**Benefits:**
- No manual typing of kanji
- Faster vocabulary lookup
- More immersive learning
- Natural reading flow

::: tip Browser Extensions
Install Yomichan (browser extension) for instant pop-up definitions when you select Japanese text!
:::

## Zoom and Pan

Get a closer look at detailed artwork or small text.

### Zoom Controls

**Mouse:**
- **Scroll wheel** - Zoom in/out
- **Click and drag** - Pan when zoomed

**Touch:**
- **Pinch** - Zoom in/out
- **Drag** - Pan around page

**Keyboard:**
- `+` or `=` - Zoom in
- `-` - Zoom out
- `0` - Reset zoom

### Zoom Retention

**Settings Option:**
- Enable "**Retain Zoom**" in reader settings
- Zoom level persists across pages
- Useful for consistent text size
- Toggle on/off as needed

**Without Retention:**
- Zoom resets on each page turn
- Default behavior
- Better for varied panel sizes

## Performance Tips

Optimize your reading experience.

### For Smooth Performance

**Good Practices:**
- Close unused browser tabs
- Use modern browsers (Chrome, Firefox, Edge)
- Ensure stable internet connection
- Avoid extreme zoom levels

**Large Volumes:**
- Vertical mode may lag with 200+ pages
- Use single/double page for very long volumes
- Browser caching improves subsequent reads

### Troubleshooting

**Pages Load Slowly:**
- Check internet speed
- Server may be processing images
- First load is slower (caching)

**OCR Text Not Appearing:**
- Ensure `.mokuro` file exists
- Re-upload volume if corrupted
- Check browser console for errors

**Zoom Feels Laggy:**
- Reduce zoom level
- Close other applications
- Try different browser
- Check CPU usage
